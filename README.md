Uso y Automatización
Este proyecto incluye herramientas de automatización mediante PowerShell para facilitar el
mantenimiento:
● Generación de índice: Se crea automáticamente un archivo SUMMARY.md.
● Limpieza: Se eliminan archivos temporales automáticamente.
● Despliegue: El repositorio está configurado para desplegarse mediante GitHub Pages.
